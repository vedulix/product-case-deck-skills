# Product Case & Deckmaking Skills

Personal Codex skill pack for product case solving, market analysis, McKinsey-style slide writing, scientific presentations, and Russian text polishing.

## Skills

- `product-case-deckmaker` - product case workflow, deck structure, Google Slides/PPTX export checklist
- `market-analysis` - MECE, SCPR, RICE, market analysis framing
- `mckinsey-style-visualization` - consulting-style visuals and executive slide composition
- `board-deck-builder` - board/investor update deck structure
- `pitch-deck-builder` - pitch deck narrative and slide flow
- `scientific-slides` - research talks, thesis defense slides, timing and visual validation
- `humanizer-ru` - Russian anti-AI / humanization editor
- `ru-text` - Russian text style, typography, UX and business writing

## Install

Copy needed folders from `skills/` into your Codex skills directory, for example:

```bash
cp -R skills/product-case-deckmaker ~/.codex/skills/
```

## Notes

This repository contains skill instructions and supporting assets only. It should not contain local credentials, tokens, or generated private working files.
