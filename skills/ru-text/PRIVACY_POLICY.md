# Privacy Policy

**Last updated:** April 9, 2026

## Overview

ru-text is a text quality plugin for Claude Code, Codex CLI, Gemini CLI, Cursor, OpenClaw, and Notion that contains only static Markdown files. It is designed to have zero impact on your privacy.

## What this plugin does NOT do

- Collect, store, or transmit any personal information
- Make any network requests to external services
- Access or modify files outside the plugin directory
- Track usage, analytics, or telemetry
- Require authentication or login
- Use cookies or similar tracking technologies

## What this plugin consists of

- Markdown reference files with text quality rules
- A plugin manifest (plugin.json)
- Slash commands defined as Markdown files

## How it works

All plugin functionality runs entirely locally within your AI coding assistant. The plugin provides knowledge that the assistant reads from local files. No data leaves your machine.
